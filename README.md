# 💰 Wealth Wellness Hub

A personal finance dashboard to monitor, track, and simulate your financial health.

---

## Features

- Connect simulated bank, brokerage, and crypto accounts  
- Add private assets (real estate, vehicles, collectibles)  
- Calculate **Wealth Wellness Score** and key financial KPIs  
- Track **net worth trend** over time  
- Visualize **portfolio composition** with interactive charts  
- Monte Carlo **wealth simulation** with inflation-adjusted projections  
- Receive AI-driven financial insights and tips  
- Export portfolio data as CSV
